Mock Media Server – API Reference

Base URL: http://localhost:3000

Endpoints

  • GET /movies – list movies
  • POST /movies – add movie (JSON body)
  • GET /series – list series
  • POST /series – add series (JSON body)
  • GET /song – list songs
  • POST /song – add movies (JSON body)

Sample JSON

{
  "id": 3,
  "title": "Dune Part Two",
  "director": "Denis Villeneuve"
}
